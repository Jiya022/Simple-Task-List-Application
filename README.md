


# 📝 Task Manager App

A simple **console-based Task Manager** built in Java.  
This application allows users to add, delete, rename, and view tasks interactively.

---

## 🚀 Features
- Add new tasks
- Delete tasks by number
- Rename existing tasks
- Show all tasks
- Exit the program gracefully

---

## ▶️ How to Run
1. Clone or download this repository.
2. Compile the Java files:
   ```bash
   javac TaskManagerApp.java TaskStore.java
   ```
3. Run the program:
   ```bash
   java TaskManagerApp
   ```

---

## 📖 Example Usage
```
--- Task Manager ---
1. Add Task
2. Delete Task
3. Rename Task
4. Show All Tasks
5. Exit
Choose: 1
Enter task name: Complete DSA assignment
Task 'Complete DSA assignment' added.
```
